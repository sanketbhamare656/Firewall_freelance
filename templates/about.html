🛡️ FIREWALL SIMULATOR- A SIMPLE PACKET FILTERING

Firewall AI is an intelligent packet filtering system that uses Machine Learning to detect and block malicious network traffic in real-time.

🎯 Project Objective

To develop an AI-powered firewall system that can automatically classify network packets as safe or malicious based on their characteristics, reducing the need for manual rule creation and improving threat detection accuracy.

⚙️ How It Works

The system uses a Random Forest Classifier trained on historical packet data, analyzing features like source/destination IP, port, protocol, and packet size to make real-time decisions.

🛠️ Technology Stack

🐍 Python 🌶️ Flask 🤖 Scikit-learn 📊 Pandas 🔢 NumPy 🌲 Random Forest 🎨 HTML5/CSS3
